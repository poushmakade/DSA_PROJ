What is linked list?

inked List is basically chains of nodes where each node contains information such as data and a pointer to the next node in the chain. It is a popular data structure with a wide range of real-world applications. Unlike Arrays, Linked List elements are not stored at a contiguous location. In the linked list there is a head pointer, which points to the first element of the linked list, and if the list is empty then it simply points to null or nothing.
OPERATIONS IN LINKED LIST

  • Insertion Operations
    • Insert at Head
    • Insert at Tail
    • Insert at a Specific Position
  • Deletion Operations
    • Delete from Head
    • Delete from Tail
    • Delete from a Specific Position
  • Traversal Operations
    • Traverse the Entire List
    • Search for an Element
  • Other Operations
    • Reverse the Linked List
    • Sort the Linked List
    • Find the Length of the List
    • Detect a Cycle in the List

APPLICATIONS OF LINKED LISTS

  • Dynamic Memory Allocation: Linked lists allow for efficient use of memory, as they can grow and shrink in size as needed.
  • Implementation of Data Structures: Linked lists are often used to implement other data structures such as stacks, queues, and graphs.
  • Undo Functionality: Linked lists can be used to keep track of changes in applications, allowing users to undo actions.
  • Polynomial Representation: Linked lists can represent polynomials in a compact way, making it easier to perform polynomial arithmetic.
  • Real-Time Applications: Linked lists can be useful in applications where frequent insertions and deletions occur, such as in memory management systems.
  • Music Playlist Management: Linked lists can maintain a dynamic playlist where songs can be added or removed easily.

           using namespace std;
           
           class LinkedList {
               int data[100];
               int next[100];
               int head;
               int free;
           
           public:
               LinkedList() {
                   head = -1;
                   free = 0;
                   for (int i = 0; i < 99; i++) {
                       next[i] = i + 1;
                   }
                   next[99] = -1;
               }
           
               void insert(int value) {
                   if (free == -1) {
                       cout << "List is full" << endl;
                       return;
                   }
                   int curr = free;
                   free = next[curr];
                   data[curr] = value;
                   next[curr] = -1;
           
                   if (head == -1) {
                       head = curr;
                   } else {
                       int temp = head;
                       while (next[temp] != -1) {
                           temp = next[temp];
                       }
                       next[temp] = curr;
                   }
               }
           
               void deleteNode(int value) {
                   if (head == -1) {
                       cout << "List is empty" << endl;
                       return;
                   }
                   if (data[head] == value) {
                       int temp = head;
                       head = next[head];
                       next[temp] = free;
                       free = temp;
                       return;
                   }
                   int temp = head;
                   while (next[temp] != -1 && data[next[temp]] != value) {
                       temp = next[temp];
                   }
                   if (next[temp] == -1) {
                       cout << "Value not found" << endl;
                       return;
                   }
                   int delNode = next[temp];
                   next[temp] = next[delNode];
                   next[delNode] = free;
                   free = delNode;
               }
           
               void display() {
                   if (head == -1) {
                       cout << "List is empty" << endl;
                       return;
                   }
                   int temp = head;
                   while (temp != -1) {
                       cout << data[temp] << " ";
                       temp = next[temp];
                   }
                   cout << endl;
               }
           };
           
           int main() {
               LinkedList list;
               int choice, value;
           
               while (true) {
                   cout << "1. Insert\n2. Delete\n3. Display\n4. Exit\nChoose an option: ";
                   cin >> choice;
           
                   switch (choice) {
                       case 1:
                           cout << "Enter value to insert: ";
                           cin >> value;
                           list.insert(value);
                           break;
                       case 2:
                           cout << "Enter value to delete: ";
                           cin >> value;
                           list.deleteNode(value);
                           break;
                       case 3:
                           list.display();
                           break;
                       case 4:
                           return 0;
                       default:
                           cout << "Invalid choice" << endl;
                   }
               }
           
               return 0;
           }
